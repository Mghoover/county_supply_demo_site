# Lumber Yard Preset

Use this overlay for lumber yards, building supply companies, hardware-driven contractor suppliers, and similar local material businesses.

This preset includes:

- A contractor-friendly homepage
- Dedicated pages for lumber, hardware, paint, locations, quote funnel, and contact
- A React-based material estimator island
- County Supply demo content as a launch-ready example

## Deployment

This site supports two deployment targets from the same codebase:

- Custom domain/root deploys default to `https://cs-lcl.demo.ordinistechnologies.com`
- GitHub Pages deploys must set `SITE_URL=https://mghoover.github.io` and `SITE_BASE=/county_supply_demo_site`

The included GitHub Actions workflow already sets those GitHub Pages values.
